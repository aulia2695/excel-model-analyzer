# Excel Data Analysis Projects

This repository contains three different cocoa farming data analysis projects.

## Projects Overview

### 1. Test 1: Cocoa Adoption Analysis (Ecuador)
- **Location**: `test-1-cocoa-adoption/`
- **Focus**: Analyzing cocoa farming practices and adoption patterns
- **Data Type**: Farm characteristics, farmer demographics, agricultural practices
- **Tasks**: Data cleaning, distribution analysis, regional comparison

### 2. Test 2: Farmer Development Plan (Ivory Coast)
- **Location**: `test-2-farmer-development/`
- **Focus**: Farmer demographics and agronomic practices analysis
- **Data Type**: Adoption variables, competence levels, production yield
- **Tasks**: Baseline analysis, progress tracking, living income surveys

### 3. Test 3: Geospatial Analysis
- **Location**: `test-3-geospatial/`
- **Focus**: Farm polygon verification and forest area identification
- **Data Type**: Kobo Collect geospatial dataset
- **Tasks**: Polygon overlap detection, forest area mapping

## Folder Structure

Each project contains:
- `raw/` - Original Excel/data files
- `cleaned/` - Processed and cleaned datasets
- `conditions/` - Analysis criteria and requirements
- `README.md` - Project-specific documentation
